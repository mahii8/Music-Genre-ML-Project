# 🎵 Music Genre Predictor AI

This project uses Machine Learning to predict the genre of a song based on its unique audio profile. 

### [View Live App Here](https://musicg.streamlit.app/)

##  How it Works
1. **Data:** Trained on a dataset of 50,000+ Spotify tracks.
2. **Model:** A **Random Forest Classifier** optimized for web deployment.
3. **Features:** Analyzes 13 key metrics including:
   - **Vibe:** Danceability, Energy, Valence.
   - **Technical:** Tempo, Key, Loudness, Mode.
   - **Texture:** Acousticness, Instrumentalness, Speechiness.

## Tech Stack
- **Language:** Python
- **ML Library:** Scikit-Learn
- **Web Framework:** Streamlit
- **Deployment:** GitHub & Streamlit Cloud
